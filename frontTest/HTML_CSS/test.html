权重等级
1.！important ，加在样式属性值后，权重值为10000
2.内联样式，如：style=""，权重值为1000
3.ID选择器，如：#content，权重值为100
4.类、伪类和属性选择器，如：.content ：hover权重值为10
5标签选择器和伪元素选择器，如：div p ：before权重值为1
6.通用选择器（*），子选择器（>），相邻选择器（+），同胞选择器（~） 权重为0 注意：
权重值相等时 后面的样式会覆盖前面的样式 权重值比较，是权重累加值的比较 eg：body .boxx{}(比重是11) > .boxx{}(权重是10)

css3 transform变换
transform:translate(x,y) 设置盒子位移,还包含<translateX,translateY,translateZ分别设置>
transform：scale(x,y)设置盒子缩放 倍数 ,还包含<scaleX,scaleY,scaleZ 分别设置>
transform:rotate(def) 设置盒子旋转 角度 ,还包含<rotateX,rotateY,rotateZ 分别设置>
transform:skew(x-angle,y-angle) 设置盒子斜切 角度（def）
transform:perspective(200px) rotateX(60deg); 设置透视距离(一般与rotateX，totateY配合使用)
transform-style:flat|preserve-3d 设置盒子是否按3d空间显示 与透视perspective配合使用
transform-origin 设置变形的中心点
backface-visibility:hidden; 设置盒子背面是否可见
